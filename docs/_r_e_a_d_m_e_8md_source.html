51 ### Pencil2D v0.5.4 beta ###
52 
53 2013-07-26

55 * ScribbleArea code refactoring.
56 * Custom keyboard shortcuts (Preferences->Shoftcuts)
57 * Interactive brush resizing. SHIFT + Mouse Drag to change brush size, CTRL + Mouse Drag for FEATHER, CTRL+SHIFT applies brush size to the eraser and temporarily selects it.

72 ### Pencil Animation v0.5 beta (Morevna Branch) - September 15th 2010
73 
74 Changes by Konstantin Dmitriev:
75 
76 * Support for some command-line options.
77  Now it's possible to specify file to open:
78  $ Pencil file.pcl
79  Export to image sequence using command-line:
80  $ Pencil file.pcl --export-sequence file.png
81 * Load dialog: Allow to choose "Any file" filter - that allows to load files without extension (old Pencil files).
82 * Fix transparency issue when exporting to png sequence
83 * Desktop integration (linux)
84 * Other minor fixes
85 
86 Changes by davidefa:
87 
88 * Export movie exports audio and video. Audio layers are mixed.
89  To select video format use an 'appropriate' filename extension ( ex: .avi, .mov, .mpg, .mp4 ... ), movie framerate is selectable ( but output codecs don't support all framerates, a safe and usual choice should be 24, 25 or 30 fps )
90 * Minor fixes in audio layers ( corrupted audio passed end of 'audio frame' ).
91 * Improved undo/redo functionality
92 * Fixed saving animation ( removed 'optimization', now saving is slower but safer )
93 * Added fps parameter in saved animations
94 * 3 'layers' of onion skinning ( configurable trough edit->preferences->tools, to remove a layer set its opacity to 0 )
95 * Added import movie ( frames are imported at the current fps, there is no check to limit the imported frames )
96 * Compiles fine with min 0.4.x on linux
97 * Other minor fixes ( moving selection on bitmap layers, moving layers... )
